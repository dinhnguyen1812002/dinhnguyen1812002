I build backend systems that are **fast**, **secure**, and **built to last**.

As a polyglot engineer, I believe strongly in choosing the right tool for each problem — not the most familiar one. Whether that's Java for enterprise-grade services, Go for high-throughput APIs, or something else entirely, I follow the requirements, not the hype.
